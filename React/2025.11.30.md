* `[개념 1]` 조건문 (if, switch)
if - 질문을 던지고 그게 참인지 거짓인지에 따라 길을 나눔( 숫자 범위를 따지거나 조건이 복잡할 때 쓰임)
`const score = 85;

if (score >= 90) {
  console.log("A학점입니다!");
} else if (score >= 80) {
  console.log("B학점입니다!"); // 여기가 실행됨
} else {
  console.log("재수강입니다..")`
  
switch - 변수 하나를 잡고 그 값이 무엇과 일치하는지 찾아감 ( 정해진 값을 고를 때 주문상태나 if-else가 많아져서 복잡한 코드가 되었을 때)
`const fruit = "사과";

switch (fruit) {
  case "사과":
    console.log("사과는 빨간색입니다.");
    break; // 여기서 멈춤! (매우 중요)
  case "바나나":
    console.log("바나나는 노란색입니다.");
    break;
  default:
    console.log("무슨 과일인지 모르겠어요.");
}`
