* `[개념 1]` 조건문 (if, switch)
if - 질문을 던지고 그게 참인지 거짓인지에 따라 길을 나눔( 숫자 범위를 따지거나 조건이 복잡할 때 쓰임)

```javascript
const score = 85;
if (score >= 90) {
  console.log("A학점입니다!");
} else if (score >= 80) {
  console.log("B학점입니다!"); // 여기가 실행됨
} else {
  console.log("재수강입니다..")
```
  
switch - 변수 하나를 잡고 그 값이 무엇과 일치하는지 찾아감 ( 정해진 값을 고를 때 주문상태나 if-else가 많아져서 복잡한 코드가 되었을 때)

```javascript
const fruit = "사과";
switch (fruit) {
  case "사과":
    console.log("사과는 빨간색입니다.");
    break; // 여기서 멈춤! (매우 중요)
  case "바나나":
    console.log("바나나는 노란색입니다.");
    break;
  default:
    console.log("무슨 과일인지 모르겠어요.");
}
```

* `[개념 4]` 화살표 함수 - 함수 표현식을 더 짧고 간결하게 쓰는 최신 문법
```javascript
let greet = function(name) {
    return name + "님 안녕하세요!";
};

// [화살표 함수] 
// 1. function 키워드 삭제
// 2. 매개변수 괄호 삭제 (하나니까)
// 3. 중괄호와 return 삭제 (한 줄이니까)
let greetArrow = name => name + "님 안녕하세요!";

console.log(greetArrow("개발자")); // 결과: "개발자님 안녕하세요!"
```
제미나이가 말하는 예시

```javascript
const numbers = [1, 2, 3];

```
* `[개념 5]` 콜백함수 - 나중에 다시 호출해줘 라는 함수
```javascript
// cook: 주방장 로봇
// menu: 햄버거 이름 (데이터)
// finishAction: "다 만들면 할 행동" (콜백 함수)
function cook(menu, finishAction) {
    console.log(menu + " 요리 시작!"); 
    finishAction(); // 시킨 일 실행
}

cook("파스타", () => {
    console.log("② 다 된 파스타를 예쁜 접시에 담습니다!");
});
```
